# Neuropave: Intelligent Infrastructure Monitoring

Neuropave is a high-performance, real-time sensor monitoring dashboard designed for infrastructure maintenance and safety. Built with Next.js, it provides a comprehensive 2D visualization of sensor networks, focusing on the San Francisco Bay Area.

## Features

- **Advanced Sensor Map**: Interactive 2D canvas-based map with smooth animations, road network overlays, and real-time sensor status indicators.
- **Real-time Analytics**: Live data aggregation and visualization of vibration, strain, temperature, and humidity sensors.
- **Intelligent Alerting**: Automated detection and notification of critical and warning levels across the network.
- **Interactive Controls**: Deep zoom, panning, and toggles for heatmaps and sensor connection lines.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technology Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com) & [Shadcn UI](https://ui.shadcn.com)
- **Visualization**: HTML5 Canvas with high-performance rendering loop
- **Icons**: [Lucide React](https://lucide.dev)
